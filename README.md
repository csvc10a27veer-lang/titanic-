# Titanic Machine Learning Project

This project performs data analysis, preprocessing, and machine learning on the Titanic dataset using Python.

## Steps Performed
- Data Understanding & Exploration
- Data Preprocessing
- Feature Encoding & Scaling
- Train-Test Split
- Logistic Regression Model Training
- Model Evaluation

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Dataset
Titanic dataset (train.csv)
